```bash
cd TOIT-TanqueCheio

```

### **2. Fazer Push da Branch Main:**
```bash
# Fazer push da branch main (onde está todo o código novo)
git push origin main

# Ou se quiser fazer merge com master primeiro:
git checkout master
git merge main
git push origin master
git push origin main
```
